Screenshots for F-Droid store listing

Required screenshots:
1. Main timer screen (stopped state)
2. Timer running with work interval
3. Timer configuration screen
4. Preset selection screen
5. Workout history screen

Screenshots should be:
- 1080x1920 pixels (or similar 9:16 aspect ratio)
- PNG format
- Named sequentially (1.png, 2.png, etc.)

To generate screenshots:
1. Install the app on a device/emulator
2. Navigate to each screen
3. Take screenshots using device screenshot function
4. Save with appropriate names in this directory
